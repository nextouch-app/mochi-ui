# Popover 气泡卡片


## Examples

<mochi-demos name="Popover"></mochi-demos>

## Web

定位气泡，支持 hover / click。

```tsx
import { Popover, Button } from '@nextouch-app/mochi-react'

<Popover title="标题" content="说明文字">
  <Button>打开</Button>
</Popover>
```

## Mobile

底部信息面板（仅点击）。

```tsx
import { Popover, Button } from '@nextouch-app/mochi-mobile'

<Popover title="标题" content="说明文字">
  <Button>打开</Button>
</Popover>
```
