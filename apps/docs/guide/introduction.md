# 介绍

**Mochi UI（麻薯 UI）** 是一套 MIT 许可、可商用的 React 组件库，由 [未来触点（Nextouch）](https://gitlab.com/nextouch) 维护。

视觉关键词：**卡通、可爱、软糯、动漫感** —— 天空蓝主色、奶油纸背景、底部色块阴影、弹性按压。

功能对齐 Ant Design 分类，移动端交互对齐 Ant Design Mobile。

## 包结构

| 包 | 用途 |
|----|------|
| `@mochi-ui/tokens` | Design Token（CSS 变量） |
| `@mochi-ui/core` | 共享类型与工具 |
| `@mochi-ui/icons` | 原创 SVG 图标 |
| `@mochi-ui/react` | Web DOM 组件 |
| `@mochi-ui/mobile` | 移动端 / H5（受限样式） |

安装方式支持 npm / yarn / pnpm / bun，见 [安装](./installation)。

详见根目录 [DESIGN.md](https://gitlab.com/nextouch/mochi-ui/-/blob/main/DESIGN.md)。
