# 介绍

**Mochi UI（麻薯 UI）** 是一套 MIT 许可、可商用的 React 组件库，由 [未来触点（Nextouch）](https://github.com/nextouch-app) 维护。

视觉关键词：**卡通、可爱、软糯、动漫感** —— 天空蓝主色、奶油纸背景、底部色块阴影、弹性按压。

按通用 / 布局 / 录入 / 展示 / 导航 / 反馈分类组织组件；Web 与 Mobile 共享 Props，移动端交互更偏触控。

## 包结构

| 包 | 用途 |
|----|------|
| `@mochi-ui/tokens` | Design Token（CSS 变量） |
| `@mochi-ui/core` | 共享类型与工具 |
| `@mochi-ui/icons` | 原创 SVG 图标 |
| `@mochi-ui/react` | Web DOM 组件 |
| `@mochi-ui/mobile` | 移动端 / H5（受限样式） |

安装方式支持 npm / yarn / pnpm / bun，见 [安装](./installation)。

详见根目录 [DESIGN.md](https://github.com/nextouch-app/mochi-ui/blob/main/DESIGN.md)。
